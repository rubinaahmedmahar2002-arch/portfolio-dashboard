# Rubina Ahmed Mahar | Analytics Command Center

Interactive portfolio dashboard built with Python and Streamlit.

## What this dashboard includes

- Command Center
- Project Galaxy
- Role Simulator
- Evidence Engine
- Skills Lab
- Experience Map
- Certificates
- Contact

## Tools

- Python
- Streamlit
- Pandas
- Plotly
- Streamlit Option Menu

## How to run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How to deploy

1. Upload these files to a GitHub repository.
2. Go to Streamlit Community Cloud.
3. Create a new app.
4. Select the repository.
5. Set main file path to `app.py`.
6. Deploy.
